# Hospital Management System

## Overview

This project is a Hospital Management System developed using Python and MySQL. It helps manage hospital operations such as patient records, doctor records, appointments, billing, and medical history.

## Features

* Add, view, update, and delete patient records
* Add, view, update, and delete doctor records
* Schedule appointments
* Generate and view patient bills
* Search patients by name
* Search doctors by specialization
* View patient medical history

## Technologies Used

* Python
* MySQL
* SQL Queries
* MySQL Connector

## Learning Outcomes

* Database connectivity using Python and MySQL
* CRUD operations (Create, Read, Update, Delete)
* Menu-driven programming
* Database management concepts
